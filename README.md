# Pupper Voice Pose Control Final Package

This clean package contains only the final files needed for voice-control testing.

Included:
- model_en/
- model_cn/
- play_pose.py
- ps4_pose_tuner.py
- voice_pose_control_bilingual.py
- poses/sit_v2.csv
- poses/final_stand_v2.csv
- poses/lay_v2.csv

Copy the model folders and Python files to ~/pupper_robot_testing, and copy the CSV files into ~/pupper_pose_tuner/poses.
